Spring Boot Actuator

<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

# Actuator settings
management.security.enabled=true
management.security.roles=ADMIN
management.endpoints.web.exposure.include=*
security.user.name=admin

/auditevents lists security audit-related events (user login/logout)
/logfile list application logs.
/loggers query logging level of our application.
/metrics details metrics of our application.
/beans returns all available beans
/conditions, builds a report of conditions around autoconfiguration.
/flyway provides Flyway database migrations.
/health health status of our application.
/heapdump list heap dump from the JVM.
/sessions lists HTTP sessions given we are using Spring Session.
/shutdown performs a graceful shutdown of the application.
/threaddump dumps the thread information of the underlying JVM.
/info list general information
/configprops list all @ConfigurationProperties beans.
/env returns the current environment
