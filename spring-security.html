Spring Security

#Add dependency
<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-security</artifactId>
</dependency>

Create Domin :

@Entity
@Table(name = "users")
public class Customer implements UserDetails

implement methods of user detail

Create Service

@Service
public class CustomerService implements UserDetailsService{
@Autowired
private CustomerRepo customerRepo;

@Override
public UserDetails loadUserByUsername(String username) throws UsernameNotFoundException {
return (UserDetails)customerRepo.getUserByUsername(username);
}

1) Extend WebSecurityConfigurerAdapter

@Configuration
@EnableWebSecurity

2) override configure method

protected void configure(AuthenticationManagerBuilder authBuilder) throws Exception {
super.configure(authBuilder.authenticationProvider(daoAuthProvider()));
}

3) Provide password encoder and user detail service to AuthProvide

@Bean
DaoAuthenticationProvider daoAuthProvider() {
DaoAuthenticationProvider provider=new DaoAuthenticationProvider();
provider.setPasswordEncoder(passwordEncoder());
provider.setUserDetailsService(userDetailsService);
return provider;
}
